Daisy - Zadig

Windows does not recognise the Daisy by default. Use Zadig to install USB drivers.

  1. Download Zadig
  2. Connect the Daisy
  3. Put Daisy into Bootloader mode
  4. Launch Zadig
  5. Options -> List All Devices
  6. Select the Daisy in the drop down list ("DFU in FS mode")
  7. Choose WinUSB as the target driver
  8. Click "Replace Driver"
